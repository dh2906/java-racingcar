---------------------------- 프로그램 흐름도 -----------------------------
1. 자동차 이름 입력

2. 입력한 이름의 유효성 검사 -> 5자 초과거나 잘못된 값(ex. 숫자)을 입력했다면 -> 예외 발생 -> 프로그램 종료

3. 시도할 횟수 입력

4. 입력한 횟수의 유효성 검사 -> 문자로 입력받았으면 -> 예외 발생 -> 프로그램 종료

5. 시도할 횟수 만큼 반복
   5-1. 0 ~ 9 범위의 난수 -> 4 이상이면 전진 / 그렇지 않으면 정지
   5-2. 진행 상황 출력

6. 가장 많이 전진한 자동차 이름 출력 -> 공동 우승인 경우 ','로 구분지어 이름 출력

---------------------------------------------------------------------------

----------------------------- 기능 -----------------------------
1. 자동차는 Car 클래스를 이용해 객체를 생성, 경주의 진행을 위한 RaceController 클래스도 생성
   RaceController 클래스에 멤버변수로 Car 클래스를 저장할 수 있도록 함

2. 자동차의 이름과 시도할 횟수에서 입력받은 값이 유효한 값인지 체크

4. 0 ~ 9 범위의 난수를 받아 전진 또는 정지를 판단하는 메소드 구현

5. 가장 많이 전진한 자동차의 이름을 우승자로 출력, 공동 우승이면 ','로 구분지어 출력
